# Linux Dotfiles
All of the dotfiles used for setting up Ubuntu Linux 20.04 programs.  

## Dependencies

* [zsh & ohmyzsh](https://github.com/ohmyzsh/ohmyzsh)
* [NeoVim](https://github.com/neovim/neovim)
* [tmux & tmuxinator](https://github.com/tmux/tmux)
